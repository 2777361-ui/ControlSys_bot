Добавить ученика

Родителя можно не выбирать. Сохраните ученика, затем в списке учеников нажмите «Пригласить родителя» — получите ссылку и отправьте её родителю для регистрации.

Отмена

Перейти к списку учеников — там у каждого ученика есть кнопка «Пригласить родителя» для получения ссылки-приглашения.
